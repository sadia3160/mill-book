# mill-book
MillBook is a digital book website for rice mill shop owners. It includes tracking and managing stock, purchases, the milling process, sales, accounts, report, etc. The main objective of MillBook is to digitalize the current notebook system of mill shops so that shop owners can easily update MillBook from anywhere—without the hassle of carrying and storing physical notebooks and performing manual calculations.
